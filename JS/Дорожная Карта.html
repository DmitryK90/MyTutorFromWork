HTML: - Основы - Структура страницы - Семантика - Формы - Emmet (дом). Теги, атрибуты, док тайп. CSS: - Селекторы(клыссы, id, наследование и тд) - Box Model - Позиционарование - Flexbox - Grid - Псевдоэлементы - Медиа запросы - Анимация (опционально) Дополнительно к CSS: - препроцессоры (SASS, Less) лучше SASS - Bootstrap (Эти три это фреймворки с классами и ид.) - Materialize CSS - Tailwind CSS - BEM(методология, структура кода) JS: - Основы языка (синтаксиси, циклы, операторы и тд.) - Взаимодействие с DOM (селекторы и тд.) - Замыкания - Стрелочные функции (синтаксис, в чем отличие от обычных функций) - Promise (хорошо разбираться) - Async Await (синтаксис для взаим. с промисами и тд.) - Event Loop - Prototype (для общего понимания, понимать теорию) - Методы массивов (forEach, map, filter и тд.) - Деструктуризация (Rest, spread и тд.) - Классы - fetch (работа с сервером, асинхронные запросы, бонусом будет поним. XMLHTPrequest(устаревшее)) Инструменты: - Git (система контроля версий) - Github, Bitbucker, Gitlab (Git репозитории) - Npm, yarn (взаимодействие со сторонними пакетами) Сборка проектов: - Webpack (топ) - Parcel (более простой и быстрый) - Snowpack Frontend (фреймворки, создание визуальных элементов): - React (бэкЭнд-Redux, Mobx) 60-70% рынка. Более низкоуровненвый, достаточно сложный. - Vue (бэкЭнд-Vuex) проще остальных. - Angular (бэкЭнд-Сервисы, NgRx) для больших проектов, но хорошо оплачиваемы. - Svelte (бэкЭнд-COntext API) перспективная, но пока не популярная, более как кампилятор. SSR (Сервер сайт рендеринг): - Next.js (React) - Nuxt.js (Vue) - Angular Universal - Sapper (Svelte) TypeScript (строго типизируемый язык, надстройка над JS, служит для помощи в разраб. в больш. прилож): ОБЯЗАТЕЛЬНО!
